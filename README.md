# Chat-Translate-App

영어 채팅을 실시간으로 번역해주는 프로그램입니다.

---

## 📦 개발 환경

### 🔧 프론트엔드

- **Node.js**: v22.0.0
- **npm**: v10.8.3
- **React**: v18.3.1
- **TypeScript**: v5.6.2

#### OCR 관련 패키지
- **Tesseract.js**: v5.1.1

### 🔧 백엔드

- **Python**: v3.10.12
- **googletrans**: v3.1.0a0

---

## 🚀 실행 방법

- **`npm install tesseract.js`**
- **`npm start`**
